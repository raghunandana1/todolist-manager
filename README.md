To-Do List Application

 
A frontend task management application designed to track and organize daily activities. 
The core logic of this application was originally written in C, and the frontend web implementation (HTML, CSS, JavaScript) was generated via AI assistance.

FEATURES:

* **Task Management:** Create, view, complete, and delete tasks.
* **Prioritization:** Assign priority levels (Low, Medium, High) to tasks.
* **Deadline Tracking:** Set due dates and view upcoming deadlines.
* **Data Organization:** Filter tasks by status (Active/Completed) and search by keywords.
* **Multiple Views:** Includes both a standard list view and a calendar grid format.
* **Local Storage:** Data is stored locally in the browser to maintain state between sessions.

Core Logic & Concept:** Written and developed in C by me.
Web UI & Implementation:** Frontend interface built with HTML, CSS, and JavaScript using AI generation.
